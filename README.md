# Windows Driver Development Studies

Repositório criado para registrar estudos introdutórios de desenvolvimento de drivers no Windows, realizados durante a leitura do livro Windows Kernel Programming, Second Edition, de Pavel Yosifovich.

O objetivo principal deste estudo foi compreender conceitos básicos de desenvolvimento em kernel mode para apoiar a criação de componentes utilizados em um projeto acadêmico de EDR desenvolvido no Trabalho de Conclusão de Curso.

## Objetivo

Este repositório tem foco educacional e foi utilizado como apoio para entender fundamentos de drivers no Windows, incluindo:

* estrutura básica de um driver;
* carregamento e comunicação com componentes em user mode;
* interação com APIs e estruturas do kernel do Windows;
* conceitos iniciais de monitoramento em baixo nível;
* base técnica para desenvolvimento de mecanismos defensivos.

## Relação com o TCC

Durante o TCC, foi desenvolvido um protótipo acadêmico relacionado à detecção e análise de comportamentos maliciosos. O estudo de drivers serviu como base para compreender como componentes em kernel mode poderiam apoiar mecanismos de monitoramento, coleta de eventos e observação de comportamento do sistema.

O projeto teve finalidade acadêmica e defensiva, com foco em aprendizado, pesquisa e segurança.

## Referência de estudo
Windows Kernel Programming, Second Edition — Pavel Yosifovich
